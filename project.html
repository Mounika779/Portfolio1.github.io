  • BMI Calculator

    • BMI stands for Body Mass Index,which is physical measurement used to assess an individual's total amount of body fat. It is a software application which avoids more manual hours that need to spend in personally calculate and find the BMI for a particular person in a single click. The BMI scores given an indirect measurement of body fat depending on the BMI value calculated individually may be Underweight,Normalweight,overweight.
    The main objective of this project is to creat a full-fledged Android application which could increase the awareness of Overweight and Underweight.
